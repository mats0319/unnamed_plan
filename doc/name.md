# 命名

项目中优先使用统一的命名，未使用之处需要特别标注

## http

1. 用户ID：userID
2. 用户名：userName
3. 昵称：nickname
4. 锁定状态：isLocked
5. 密码：password
6. 权限等级：permission
7. 创建人：createdBy
8. 分页信息
    1. 每页条数：pageSize
    2. 当前页码：pageNum （当前页码从1开始）
    3. 符合条件的数据总条数：total
9. 请求的发起者ID：operatorID
10. 执行结果：isSuccess
11. 文件ID：fileID
12. 文件名：fileName
13. 扩展名：extensionName
14. 文件大小：fileSize
15. 是否公开：isPublic
16. 文件：file
17. 访问路径：fileURL
18. 创建时间：createdTime
19. 更新时间：updateTime

## CRUD

### 数据库

1. 插入：insert
2. 更新：update
3. 查询：query
4. 删除：delete

### 前、后端程序

1. 新增：create
2. 修改：modify
3. 查询：list
4. 删除：delete
